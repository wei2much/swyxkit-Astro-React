This is a sample blog post to demonstrate the new **SwyxKit** built with Astro + React + Tailwind!

## Features

- 🚀 **Astro v5** for lightning-fast static sites
- ⚛️ **React v19** for interactive components
- 🎨 **Tailwind v4** for utility-first styling
- 📝 **MDX** for rich content authoring
- 🔍 **Shiki** for beautiful syntax highlighting
- 📱 **Responsive** design that works everywhere

## Code Example

Here's some beautiful syntax-highlighted code:

```typescript
interface BlogPost {
  title: string;
  slug: string;
  date: Date;
  category: PostCategory;
  tags: string[];
  featured?: boolean;
}

export function createPost(data: BlogPost): BlogPost {
  return {
    ...data,
    date: new Date(data.date)
  };
}
```

## Getting Started

1. Clone the repository
2. Install dependencies with `pnpm install`
3. Start the development server with `pnpm dev`
4. Begin writing amazing content!

This template is a faithful port of the original SvelteKit version, maintaining feature parity while leveraging modern Astro capabilities.
