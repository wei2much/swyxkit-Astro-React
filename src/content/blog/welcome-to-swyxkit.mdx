## Getting Started

1. Clone this repository
2. Install dependencies with `pnpm install`
3. Start the development server with `pnpm dev`
4. Create your content:
   - **Blog posts**: Add `.mdx` files to `src/content/blog/`
   - **Pages**: Add `.mdx` files to `src/content/pages/`
5. Begin writing amazing content in MDX!

This template maintains complete feature parity with the original SvelteKit version while leveraging Astro's unique strengths for better performance and developer experience.

## Content Management

Unlike the original SwyxKit that used GitHub Issues as a CMS, this version uses local MDX files:

- **Blog posts** go in `src/content/blog/your-post.mdx`
- **Static pages** go in `src/content/pages/your-page.mdx`
- Full MDX support with React components
- Frontmatter for metadata and configuration

> **Note**: All original SwyxKit features have been preserved, including dark mode, search, RSS, and responsive design.
